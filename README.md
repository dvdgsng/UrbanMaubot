# urban

A [maubot](https://github.com/maubot/maubot) to fetch definitions from [Urban Dictionary](https://www.urbandictionary.com/). Returns a specified or random entry.

## Usage

    !ud [term] [index]

## Examples

    !ud
    !ud foo
    !ud foo 2
